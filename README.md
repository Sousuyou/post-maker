# 営業投稿メーカー（Bar Soutsu 社内ツール）

毎日のLINE投稿文を、選ぶ・入れるだけで自動で組み立てる入力補助ツールです。

- 公開URL（予定）: https://sousuyou.github.io/line-post-maker/
- 方式: **入力補助型**（AIサーバ不要・完全無料・オフライン動作）
- 言い回しは「シード（乱数の種）」で決まり、「言い回しを変える」ボタンで一新されるため、毎日コピペしても文がかぶりません。

## 機能
- 文章の雰囲気を3トーンから選択（落ち着いた大人／フレンドリー／にぎやか）
- 投稿に入れる段落を入/切（季節・天気・曜日の挨拶／おすすめ酒・入荷・限定／営業時間・空席・予約／イベント告知／締めの一言／ハッシュタグ）
- 日付から曜日・季節を自動判定
- 店舗プロフィール（店名・営業時間・電話・定番タグ）は端末に自動保存
- ワンタップでコピー、文字数カウント

## 構成ファイル
- `index.html` … 画面
- `app.js` … 文章生成ロジック（素材プール＋シード固定の乱数）
- `styles.css` … デザイン（共通土台＋専用スタイル）
- `boot.js` … Service Worker登録
- `manifest.json` / `service-worker.js` / `.nojekyll` … PWA構成
- `assets/icon.svg` … アプリアイコン
- `assets/og-image.png` … SNS共有画像（`og-source.svg`から生成）

## 共通の仕組み
- 上部ナビは `top/nav.js`、色・ダークモードは `top/theme.css`（共通の元）を読み込み。
- Service Worker更新方針: HTML/データはネットワーク優先、CSS/JSはキャッシュ優先（変更時は `service-worker.js` の `CACHE` 版数を上げる）。

詳しい開発標準は親フォルダの `_管理/開発ルール.md` を参照。
